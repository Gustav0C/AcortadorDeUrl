🎯 PASOS PARA CONECTAR CON GITHUB:

1. Ve a https://github.com/new
2. Nombre: fis-url
3. Descripción: Un acortador de URLs moderno con interfaz minimalista y códigos QR
4. Público ✅
5. NO marcar README, .gitignore, ni license
6. Crear repositorio
7. Copiar la URL que te aparece (https://github.com/TU-USUARIO/fis-url.git)
8. Ejecutar en la terminal:

git remote add origin https://github.com/TU-USUARIO/fis-url.git
git push -u origin main

¡Una vez hecho esto, tu código estará en GitHub!
